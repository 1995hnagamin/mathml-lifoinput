命令
####

数 (``<mn>`` 要素)
**********************

例
======

入力. ::

    1
    1.234

出力. ::

    <mn>1</mn>
    <mn>1.234</mn>

識別子 (``<mi>`` 要素)
**********************

例
======

入力. ::

    mi:"sin"
    mi:"x"
    x

出力. ::

    <mi>sin</mi>
    <mi>x</mi>
    <mi>x</mi>

演算子，囲い文字，区切り文字，アクセント (``<mo>`` 要素)

入力. ::

    +
    -

出力. ::

    <mo>+</mo>
    <mo>-</mo>
