プレゼンテーションマークアップ
********************************************

トークン要素
============================================

識別子 (``<mi>`` 要素)
--------------------------------------------

例
^^^^

入力. ::

    mi "sin"
    mi "x"
    x

結果. ::

    <mi>sin</mi>
    <mi>x</mi>
    <mi>x</mi>

数 (``<mn>`` 要素)
--------------------------------------------

例
^^^^

入力. ::

    1
    1.234

結果. ::

    <mn>1</mn>
    <mn>1.234</mn>

演算子，囲い文字，区切り文字，アクセント (``<mo>`` 要素)
-------------------------------------------------------------------

例
^^^^

入力. ::

    +
    -

結果. ::

    <mo>+</mo>
    <mo>-</mo>

一般的なレイアウトスキーマ
============================================

水平方向にグループ化された部分式 (``<mrow>`` 要素)
-------------------------------------------------------------------

例
^^^^

入力. ::

    1
    +
    2
    mrow 3

結果. ::

    <mrow><mn>1</mn><mo>+</mo><mn>2</mn></mrow>

べき乗根 (``<msqrt>``, ``<mroot>`` 要素)
--------------------------------------------

``msqrt`` はスタックの先頭要素を1つポップし， それを ``<msqrt>`` 要素で囲んだ要素をプッシュします．

``mroot`` はスタックの先頭要素を2つポップし，それらを ``<mroot>`` 要素で囲んだ要素をプッシュします．

例
^^^^

入力. ::

    10
    msqrt
    2
    3
    mroot

結果. ::

    <msqrt><mn>10</mn></msqrt>
    <mroot><mn>2</mn><mn>3</mn></mroot>

添字 (``<msup>`` 要素)
--------------------------------------------

例
^^^^

入力. ::

    a
    2
    msup

結果. ::

    <msup><mi>a</mi><mn>2</mn></msup>

XML 属性
